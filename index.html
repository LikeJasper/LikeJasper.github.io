services

Get in touch if you need a website or you'd like an existing website improved! As I'm building up a portfolio right now I can offer very low prices.

I speak Python and JavaScript, so I offer back end web development using Django (a web framework for Python) and Meteor (a web framework for JavaScript). I can also work with PHP unless your site is unusually complex.

On the front end, I use HTML5 and CSS3 (usually compiled from SCSS), together with jQuery and custom JavaScript. I have a preference for the Materialize design framework, but I also work with Bootstrap.

All my projects are under version control with Git and I work with repositories on Github and Bitbucket.

I keep an eye on developments in web design and I have a collection of resources to help with your design decisions if you don't want to hire a separate web designer.

about

I moved to Berlin in summer 2014, before which I lived in London. Previously I studied Philosophy and Psychology at the University of Oxford, completing the two-year BPhil master's degree in Philosophy in 2013.

I built my first interactive website in 2011 with PHP (a very simple time-difference calculator), and really enjoyed it. Since then I have explored a wider range of web technologies and I have been working on inculcating a set of best practices in my programming. To that end I have become increasingly interested in test-driven development, in particular behaviour-driven development.
