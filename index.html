services

I speak Python and JavaScript, so I offer back end web development using Django (a web framework for Python) and Meteor (a web framework for JavaScript). I can also work with PHP if your site is not unusually complex.

On the front end, I use HTML5 and CSS3, together with jQuery as well as custom JavaScript. I have a preference for the Materialize design framework, but I am also comfortable with Bootstrap.

I'm not a trained designer, but I keep an eye on developments in web design and I have a collection of resources to help you with your design decisions if you don't want to hire a separate web designer.
